# Camoufox Worker

Anti-detection browser automation service powered by [Camoufox](https://github.com/daijro/camoufox) — a Firefox fork with C++ level fingerprint spoofing.

## Why Camoufox over Chromium + Stealth?

| Aspect | Chromium + JS Stealth | Camoufox |
|--------|----------------------|----------|
| Spoofing | JavaScript injection (detectable) | C++ source patches (native) |
| Protocol | CDP (widely fingerprinted) | Juggler (not targeted) |
| WebGL/Audio/Canvas | JS override | Native C++ managers |
| Headless detection | Partial JS fix | Binary-level patches |
| navigator.webdriver | JS `defineProperty` | Removed from source |

## API

Drop-in replacement for `cf-browser-worker`. Same endpoints, same request format.

### POST /execute

```json
{
  "proxy": { "server": "http://...", "username": "...", "password": "..." },
  "timezone": "America/New_York",
  "os": "windows",
  "ttl": 300000,
  "captcha": { "provider": "2captcha", "apiKey": "..." },
  "steps": [
    { "action": "goto", "params": { "url": "https://example.com" } },
    { "action": "screenshot", "params": { "fullPage": true } }
  ]
}
```

**New parameters (vs cf-browser-worker):**
- `os` — Force fingerprint OS: `"windows"`, `"macos"`, `"linux"`. Random weighted if omitted.
- `fingerprint` — Custom fingerprint overrides (timezone, screen, webgl, etc.)
- `stealth` — Removed (Camoufox handles this natively)
- `userAgent` — Removed (generated by fingerprint engine to match OS/browser)

### Other endpoints

- `GET /health` — Health check (returns `{ ok, engine: "camoufox", sessions }`)
- `GET /sessions` — List active sessions
- `GET /sessions/:id` — Get session details
- `DELETE /sessions/:id` — Close session

## Build & Deploy

```bash
docker build -t camoufox-worker .
docker run -p 3002:3002 --shm-size=2g camoufox-worker
```

## Architecture

```
Request → Express → createSession()
                      ├── generateFingerprint() → CAMOU_CONFIG env var
                      ├── firefox.launch({ executablePath: camoufox-bin })
                      ├── context.addInitScript(per-context setters)
                      └── page ready

Steps → executeStep() → Playwright Firefox API (same as Chromium)
```

The Camoufox binary reads `CAMOU_CONFIG` at startup and configures 92 properties at the C++ level. Per-context properties (audio seed, canvas seed, font spacing) are set via self-destructing `window.setXxx()` functions that remove themselves after first call.
